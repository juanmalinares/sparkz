# **App Name**: QuizWhiz Kids

## Core Features:

- User Profile Creation: Allow users to create a profile by setting a username; persist it using synched database accesible from everywhere.
- Interactive Quizzes: Provide a quiz-taking interface, showing one question at a time in multiple choice format.
- Score Tracking: Display the quiz results and overall high score in the session. Provide links to retry or select another quiz.
- AI-Powered Feedback: Use generative AI to generate simple and reinforcing feedback/explanations when the answer is either correct or incorrect. The feedback will show immediately on question completion, if AI determines it's relevant to display such feedback for this particular question.
- AI Quiz Creator (Admin): Design a page only accesible to 'JML' with the Quiz Creator. An admin (JML) will have access to an AI Quiz Creator tool that will generate new quizzes by processing user input and optional images, creating diverse question formats based on the instructions.
- Performance Statistics Dashboard: Design a dedicated page displaying student performance statistics: include the high score, the recent results of quizzes and study paths.
- Student Tutorial Guide: Create a comprehensive tutorial guide that explains all concepts and how to utilize all features to its maximum. A button to flag questions as confusing or wrong. And get a substitute question. That question should be flag to the admin.

## Style Guidelines:

- Primary color: Vibrant orange (#FF9327) to represent energy and activity, drawing from the color palette specified in the request.
- Background color: Very light lilac (#F5F2FF), near-white, to ensure good readability and create a friendly and inviting atmosphere, derived from the request.
- Accent color: Deep violet (#9448CF), used sparingly for interactive elements and key information, complementing the primary color and providing visual contrast as mentioned in the request.
- Body and headline font: 'PT Sans' (sans-serif) to give a mix of contemporary style and a bit of warmth.
- Use clean, modern icons with rounded edges to visually represent each feature of the quizzes.
- A card-based layout using Material Design, providing depth and organization for each question, enhancing ease of use, which were a request in the documentation.
- Subtle, cheerful animations for correct answers and score updates, making the app dynamic and keeping students interested. Simple fades for page transitions.
